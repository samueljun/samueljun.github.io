Amazon Price Checker

Receive notifcations when your items are on sale

Amazon Price Checker is a website that will send users a notification when items on his or her wishlist is currently on sale or have hit a new historical low. I conceived the idea for the website from my need to track the prices of the items on my Amazon wishlist. The website will also allow users to add to their tracked items through direct URLs of the items. The notification system will include email and text messages with a price history so that the user can make an informed decision.

The website is still in the conceptual design phase. Currently the website is running on my netbook web server described below.

KnowPlace [GitHub]

There's KnowPlace like home...

KnowPlace is a web application primarily aimed at do-it-yourself (DIY) enthusiats who want to do home automation. KnowPlace provides a web service where DIY enthusiasts who want to do home automation can easily use the included libraries to connect to the website to do home automation. The user will use the KnowPlace libraries to connect the the KnowPlace server and the user will be able to interact with his/her device using the website. The site will feature automated events, manual view and control, and a future mobile application.

The project is still early in development but full control of light switches and variable fan speed is implemented. The website employs Apache Tomcat for servlets and JSP, Maven, PostgreSQL, jQuery, AJAX.

Netbook Web Server

As a personal project, I converted my netbook into a psuedo portable web server. My motivation for the project was to apply my web developer skills and networking knowledge into something that was fun and also useful. Currently the netbook is a LAMP stack using Ubuntu Server 12.04 LTS. Along with the web server is a Mumble server for a dedicated VOIP solution when gaming my friends.

Future plans include a Plex media server as well as a cloud storage solution. If you have any suggestions for cool things I can do with a personal web server, please send me an email! I'm looking for more ideas!
