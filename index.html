Amazon Price Checker

Receive notifications when your items are on sale

Amazon Price Checker is a website that will send users a notification when items on his or her wish list is currently on sale or have hit a new historical low. I conceived the idea for the website from my need to track the prices of the items on my Amazon wish list. The website will also allow users to add to their tracked items through direct URLs of the items. The notification system will include email and text messages with a price history so that the user can make an informed decision.

The website is still in the conceptual design phase. Currently the website is hosted on my netbook web server described below.

KnowPlace [GitHub]

There's KnowPlace like home...

KnowPlace is a web application primarily aimed at do-it-yourself (DIY) enthusiats who want to do home automation. KnowPlace provides a web service where DIY enthusiasts who want to do home automation can easily use the included libraries to connect to the website to do home automation. The user will use the KnowPlace libraries to connect the the KnowPlace server and the user will be able to interact with his/her device using the website. The site will feature automated events, manual view and control, and a future mobile application.

The project is still early in development but full control of light switches and variable fan speed is implemented. The website employs Apache Tomcat for servlets and JSP, Maven, PostgreSQL, jQuery, AJAX.

Netbook Web Server

As a personal project, I converted my netbook into a web server. My motivation for the project was to apply my web developer skills and networking knowledge for something fun and useful for both me and my roommates. Currently the netbook is setup with a LAMP stack using Ubuntu Server 12.04 LTS. Along with the web server is a Mumble server for a dedicated VOIP solution when gaming with my friends. The most recent addition to the server is a Plex Media Server for a dedicated DLNA media solution. The main motivation for the Plex Media Server was to have the ability to stream music and movies instead of having to use of storage space and sync libraries. The Plex Media Server also makes it convenient to stream music straight to the living room PS3. Later down the line, I hope to set up a more powerful media PC with a NAS.

Future plans includes a personal cloud storage solution akin to Dropbox. If you have any suggestions for other cool or nifty things I can do with a personal web server, please let me know! I'm looking for more ideas!
